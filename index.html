Roadmap
Future Improvements
01
EfficientNetV2 Backbone
Upgrade to EfficientNetB0/V2 for better accuracy-to-parameter ratio on medical images.
02
Grad-CAM Explainability
Visualize which regions in the X-ray drive predictions to build clinician trust.
03
Larger Validation Set
Current val set is only 16 images — expand to 500+ for reliable hyperparameter tuning signal.
04
Class Weighting
Apply class weights (72.83% vs 27.17% imbalance) to improve specificity without hurting recall.
05
FastAPI Deployment
Package the DeepLearningPredictor class into a REST API with Docker containerization.
06
Ensemble Models
Combine MobileNetV2, DenseNet, and ResNet50 predictions for a more robust system.
